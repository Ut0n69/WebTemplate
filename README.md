# static-web-template

This is a static web site template.

## Usage

```
$ git clone https://github.com/Ut0n69/static-web-template.git
$ cd static-web-template
$ yarn
$ yarn serve
```

## What's in the template?

- browser-sync
- browserify
- babelify
- babel(latest)
- sass(SCSS)
- postcss(autoprefixer, mqpacker, declaration-sorter)
- ejs
- gulp
- eslint(airbnb)
